    lazygit#

    用于 git 命令的简单终端界面

    资料#

    • lazygit

    使用#

    add

    # 在【2】区域

# 1、
# 按空格（切换暂存）

# 2、
# 按 v 后，然后按上下键选中多条，然后按空格（切换暂存）

# 3、
# 按 a 后，全部文件（切换暂存）

    commit

    # 在【2】区域

# 按 c 会弹出输入框要求填入提交信息，输入后 enter 即可提交已经暂存的更改；如果是需要添加详细可以使用 C

    reset

    pull

    # 在任意位置都可通过 p 进行代码拉取，会拉取左栏第三栏中的 Local Branches 的分支列表中的带星号的分支的远程分支代码；

    push

    # 在任意位置都可通过 P 进行代码推送，会推送代码到左栏第三栏中的 Local Branches 的分支列表中的带星号的分支的远程分支中；
