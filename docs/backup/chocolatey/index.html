    Chocolatey#

    Windows 程序包管理器，现代软件自动化

    参考#

    • The Package Manager for Windows

    命令#

    # 查找 xxx 安装包
choco search xxx

# 查看 xxx 安装包信息
choco info xxx

# 安装 xxx 软件
choco install xxx

# 升级 xxx 软件
choco upgrade xxx

# 卸载 xxx 软件
choco uninstall xxx
